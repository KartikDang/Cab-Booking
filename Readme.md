Instructions to Run the code

First Step:
```
$ git clone https://github.com/KartikDang/Cab-Booking
$ cd Cab-Booking
```
Second Step:
Run the sql files added to the github repository

Third Step:
```
$ cd backend
$ yarn install
$ cd ..
$ cd client
$ yarn install
```

Fourth Step:
Change the sql connection particulars in the index.js file in backend

Fifth Step:
```
$ cd ..
$ cd backend
$ yarn start
$ cd client
$ yarn start
```
